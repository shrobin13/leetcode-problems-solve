# LeetCode Problem Solving Journey

This repository documents my step-by-step journey of solving LeetCode problems using **Data Structures and Algorithms (DSA)**.

I am practicing in a structured way:
- Start with a topic (Array, String, Sliding Window, etc.)
- Solve problems from easier to more challenging patterns
- Build strong fundamentals first, then improve speed and optimization

## What I Am Doing

I am solving LeetCode problems topic-by-topic to develop:
- Problem-solving intuition
- Pattern recognition
- Clean coding habits
- Interview-ready DSA skills

Each solution is part of a continuous learning process where I focus on:
- Understanding the problem clearly
- Choosing an appropriate approach
- Writing correct and readable code
- Improving time and space complexity when possible
